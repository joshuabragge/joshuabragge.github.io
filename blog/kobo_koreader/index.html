Install KO Reader on kobo aura one

July 31, 2018 Installations

Here are steps I followed to install ko reader on kobo aura one.

Install KSM

  • KSM 9 is the latest build at the time of this installation. Downlod it here

  • Extract the zip, you will have the following inside it.

    kbmenupngs
KoboRoot.tgz

  • Now connect kobo to laptop using usb and place kbmenupngs in the root of the kobo and reboot the device by disconnecting it.

  • Place KoboRoot.tgz in .kobo folder and reboot. Now you can see ksm menu after reboot. You can click start nickel to move to kobo original menu.

  • You have succesfully installed KSM.

Install KO Reader

  • Download the build of kobo ko reader here
  • unzip and copy koreader folder into .adds directory of kobo device by connecting it.
  • Reboot the device and now you should see start KOReader in the KSM menu.

You are done with the installation !!
